# 실행 방법
# 1. '실행하기.bat' 파일을 더블 클릭하여 실행하세요.
# 2. 브라우저가 자동으로 열립니다.
# 3. 사방넷 통합 주문내역 엑셀 파일을 업로드하세요.
# 4. 업체를 선택하고 '변환 및 압축파일 생성' 버튼을 누르세요.

# 주요 기능
# - 28개 업체별 주문서 자동 변환
# - 엑셀 파일 내 AutoFilter(필터/정렬) 기능 내장
# - 동송농협 전용 텍스트(TSV) 파일 자동 생성
# - 보류 데이터 관리 (저장/병합/삭제)
# - 청수굴비 주문 분할 처리
