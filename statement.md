## 1. 프로젝트 개요

**365 건강농산 주문서 변환기**는 사방넷(Sabangnet) 통합 주문내역 엑셀 파일을 업체별 고유 양식으로 자동 변환하는 Streamlit 웹 애플리케이션입니다.

| 항목 | 내용 |
|------|------|
| **기술 스택** | Python 3.x, Streamlit 1.53.1, Pandas 2.3.3, OpenPyXL 3.1.5, SQLite3 |
| **실행 방식** | `streamlit run convert.py` 또는 `실행하기.bat` (포터블 Python 포함) |
| **대상 업체 수** | 28개 업체 (nh_list) |
| **양식 템플릿** | 29개 파일 (`data/target_form/`) |
| **배포 형태** | 포터블 Python 번들 (`python/` 디렉토리) |

## 2. 시스템 아키텍처

```
[사용자] → [Streamlit UI (convert.py) — 4탭 구조]
            ├── 📋 주문서 변환 탭 (기존 핵심 기능)
            │     ├── 엑셀 파일 업로드
            │     ├── 상품약어 기준 업체 분류 (map.py → sort_data)
            │     ├── 업체 선택 (체크박스 그리드)
            │     ├── 컬럼 매핑 → 고정값 → 복제 → 커스터마이징 → 엑셀 생성
            │     └── ZIP 다운로드 + 로컬 저장 (data/converted/)
            ├── 🏭 업체 관리 탭 (vendor_manager.py)
            │     └── 업체 CRUD + 양식 업로드 + 키워드 설정
            ├── 🏷️ 키워드 관리 탭 (keyword_manager.py)
            │     └── 키워드 조회/검색/추가/삭제/일괄수정
            └── 🔄 주문 변환 매핑 탭 (order_mapping.py)
                  └── 원본↔업체 칼럼 매핑 + 분류 기준 설정
```
