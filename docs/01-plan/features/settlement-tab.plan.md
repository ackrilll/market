# Plan: 정산 탭 기능 추가

> **Created**: 2026-04-12
> **Status**: Draft

---

## 1. 개요

기존 "주문서 변환" 결과를 기반으로 **거래처별 정산** 기능을 추가한다.
주문 데이터에 `농협 매입가`와 `누계` 칼럼을 자동으로 추가하여 정산용 Excel 파일을 생성한다.

## 2. 현재 상태

- Streamlit 앱에 4개 탭 존재: 주문서 변환, 업체 관리, 매핑 관리, 변환 미리보기
- 주문서 변환 탭에서 사방넷 통합 주문 Excel → 업체별 Excel 파일 생성
- `data/정산/참조/` 폴더에 상품리스트 참조 파일 존재 (365건강농산 시트의 `매입가` 사용)

## 3. 목표

사이드바에 **"정산"** 메뉴 탭을 추가하여:

1. 주문서 변환 결과(또는 동일 형식 파일)를 입력으로 받음
2. `품번코드(필수)` 기준으로 참조 파일에서 `매입가` 조회
3. `농협 매입가` 칼럼 삽입 (수량 뒤)
4. `누계` 칼럼 삽입 (`수량 × 농협 매입가`)
5. 정산 완료된 Excel 파일 다운로드 제공

## 4. 입출력 데이터 구조

### 4.1 Input (18열)
| 칼럼 | 설명 |
|------|------|
| 쇼핑몰 | 판매 채널명 |
| 주문번호 | 주문 고유번호 |
| 수령인 | 받는 사람 |
| 상품약어 | 상품 분류 약어 |
| 수량 | 주문 수량 |
| 수령인 전화번호 | - |
| 수령인 휴대폰 | - |
| 우편번호 | - |
| 수령인주소 | - |
| 배송메세지 | - |
| 구매자 | - |
| 구매자 휴대폰 | - |
| . | 구분자 (빈 칼럼) |
| 사방넷 주문번호(필수) | - |
| 송장번호(필수) | - |
| 택배사(필수) | - |
| 택배사코드(필수) | - |
| 품번코드(필수) | 매입가 조회 키 |

### 4.2 Output (20열)
Input과 동일하나 `수량` 뒤에 2개 칼럼 추가:
- **농협 매입가**: 참조 파일에서 `품번코드` 기준 `매입가` 조회
- **누계**: `수량 × 농협 매입가`

### 4.3 참조 파일
- **파일**: `data/정산/참조/0330_365건강농산 및 직코드 상품리스트 (2).xlsx`
- **시트**: `365건강농산` 시트만 사용
- **조회 키**: `품번코드` → `매입가`
- **관리**: 미리 저장된 파일 자동 로드 + 사용자 업로드로 교체 가능

## 5. 핵심 로직

```
1. 사용자가 정산용 입력 파일(변환 결과) 업로드
2. data/정산/참조/ 폴더에서 상품리스트 자동 로드 (또는 사용자가 업로드하여 교체)
3. 365건강농산 시트에서 품번코드→매입가 매핑 테이블 생성
4. 입력 파일의 각 행에서:
   a. 품번코드(필수) 값으로 매입가 조회
   b. 농협 매입가 = 조회된 매입가
   c. 누계 = 수량 × 농협 매입가
5. 수량 칼럼 뒤에 농협 매입가, 누계 칼럼 삽입
6. 빈 행(업체 구분 행)은 그대로 유지
7. 결과 Excel 파일 생성 및 다운로드 제공
```

## 6. 구현 계획

### 6.1 파일 구조
| 파일 | 작업 | 설명 |
|------|------|------|
| `settlement_tab.py` | **신규** | 정산 탭 UI 및 로직 |
| `convert.py` | **수정** | 사이드바 메뉴에 "정산" 추가 + import |

### 6.2 settlement_tab.py 주요 함수
- `render_settlement_tab()`: 메인 UI
  - 입력 파일 업로드 (st.file_uploader)
  - 참조 파일 로드/교체 UI
  - 정산 처리 버튼
  - 결과 미리보기 (st.dataframe)
  - 다운로드 버튼 (st.download_button)
- `load_reference_data(file)`: 참조 파일에서 품번코드→매입가 딕셔너리 생성
- `process_settlement(df, price_map)`: 입력 데이터에 농협 매입가/누계 칼럼 추가

## 8. 예외 처리

| 상황 | 처리 |
|------|------|
| 품번코드가 참조에 없음 | 경고 표시, 농협 매입가/누계를 빈값으로 처리 |
| 수량이 비어있거나 숫자 아님 | 해당 행 누계 빈값 처리 |
| 빈 행 (업체 구분선) | 그대로 유지, 정산 로직 스킵 |
| 참조 파일이 없음 | 업로드 요청 안내 |
| 참조 파일 시트명 불일치 | 오류 메시지 표시 |
